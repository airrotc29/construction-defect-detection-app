# ACE TECH 하자관리 시스템

집합건물(공동주택) 하자적출을 위한 브라우저 기반 도구입니다. 서버 없이 정적 파일(`index.html`)만으로
동작하며, GitHub Pages로 바로 배포할 수 있습니다.

## 주요 기능

- 동/호수(전유부·공용부) 목록 엑셀 업로드 / 양식 다운로드
- 하자 등록 시 **연차별 하자담보책임 기준**(집합건물법 시행령 제5조, 법 제9조의2)에 따라
  기산일(전유부: 키불출일 / 공용부: 사용승인일) 기준 제척기간 만료일과 잔여일수(D-day) 자동 계산
- 하자 사진·동영상을 **브라우저에서 지정한 PC 폴더**에 동/호수별로 자동 정리 저장
  (File System Access API — 데스크톱 Chrome/Edge 전용)
- 하자보수 완료 후 결과 사진/동영상 추가 업로드
- 건설사(시공사) 앞 **하자보수 이행 청구서(내용증명) 초안** 생성
- 호수별 상세 보고서 및 전체 집계 보고서(인쇄/PDF), 엑셀 집계 내보내기
- 데이터 백업(JSON) 내보내기/불러오기

## 사용 방법

`index.html`을 데스크톱 Chrome 또는 Edge 브라우저로 열면 바로 사용할 수 있습니다.
GitHub Pages로 배포하려면 저장소 설정에서 Pages 소스를 이 브랜치의 루트(`/`)로 지정하세요.

데이터는 브라우저(IndexedDB)와 사용자가 직접 연결한 PC 폴더에만 저장되며, 별도의 서버로 전송되지 않습니다.

## 면책 고지

본 도구는 하자적출·기록 및 제척기간 계산을 보조하기 위한 실무 도구이며, 특정 사건에 대한 법률 자문이
아닙니다. 실제 청구·소송 진행 전 최신 법령과 전문가 자문을 확인하시기 바랍니다.
