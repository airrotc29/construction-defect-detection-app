# ACE TECH 하자관리 시스템

집합건물(공동주택) 하자적출을 위한 브라우저 기반 도구입니다. 정적 파일(`index.html`)과 하나의
Cloudflare Worker(`server/`)만으로 동작하며, GitHub Pages로 배포할 수 있습니다.

## 주요 기능

- 관리자 로그인 → 사업소별 아이디/비밀번호 발급, 사업소 계정 목록·등록 하자 건수 확인
- 사업소 계정 로그인 → 현장 정보 입력, 하자 목록 일괄 등록
- 사업장마다 다른 하자 목록 서식(엑셀·한글·PDF 등)을 그대로 가져오기 — 공용부분/전유부분 각각
  엑셀 업로드 또는 표 붙여넣기(Ctrl+V)로 일괄 등록, 매핑 없이 동/호수 열 자동 인식
- **연차별 하자담보책임 기준**(집합건물법 시행령 제5조, 법 제9조의2) 안내 — 기산일(전유부: 인도일/
  키불출일 / 공용부: 사용검사일·사용승인일) 기준 존속기간(제척기간) 분류를 참고 자료로 제공

## 사용 방법

먼저 `server/README.md`를 따라 백엔드(Cloudflare Worker)를 배포하고, `index.html` 상단의
`API_BASE` 값을 배포된 Worker 주소로 바꾸세요. 이후 GitHub Pages로 배포하려면 저장소 설정에서
Pages 소스를 이 브랜치의 루트(`/`)로 지정하세요.

로그인한 사업소 계정의 입력 데이터는 이 저장소의 `data/` 폴더(비공개 API를 통해서만 접근 가능)에
보관되며, 브라우저에는 로그인 세션 토큰만 저장됩니다.

## 면책 고지

본 도구는 하자적출·기록을 보조하기 위한 실무 도구이며, 특정 사건에 대한 법률 자문이 아닙니다.
실제 청구·소송 진행 전 최신 법령과 전문가 자문을 확인하시기 바랍니다.
